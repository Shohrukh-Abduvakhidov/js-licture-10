## BOM (Browser Object Model) дар JavaScript

## BOM (Browser Object Model) як модели объектоӣ мебошад, ки браузерро муаррифӣ мекунад. Он ба барномасозон имконият медиҳад, ки бо компонентҳои муҳити браузер кор кунанд. Ин модел ба DOM монанд аст, аммо ба унсурҳои браузерӣ, на ба мундариҷаи саҳифа, марбут аст.

### Қисмҳои асосии BOM

1. **`window` (тиреза)**  
   `window` объекти асосии BOM мебошад, ки тамоми дигар объектҳои BOM ҳамчун зеркомпонентҳои он шомиланд.
   - Ҳар як объекти глобалӣ ё функсия дар JavaScript ба объекти `window` тааллуқ дорад.

# Фарқият байни DOM ва BOM

- **DOM:** Бо сохтор ва мазмуни саҳифа кор мекунад (унсурҳои HTML).
- **BOM:** Бо муҳити браузер (тиреза, URL, history) кор мекунад.

# BOM яке аз қисмҳои муҳими JavaScript мебошад, ки барои эҷоди вебсайтҳои интерактивӣ истифода мешавад.

# DOM (Document Object Model) дар JavaScript

## DOM (Document Object Model) дар JavaScript як интерфейси сохторӣ мебошад, ки вебсаҳифаҳои HTML ва XML-ро ба объекти динамикӣ табдил медиҳад. Ин ба барномасозон имконият медиҳад, ки бо мундариҷа, сохтор ва тарзи саҳифа тавассути JavaScript кор кунанд.

## Маълумоти асосӣ дар бораи DOM

### 1. Сохтори дарахтмонанд

DOM вебсаҳифаро ҳамчун дарахт (tree structure) муаррифӣ мекунад, ки дар он ҳар як унсури HTML ё тег як нод (node) мебошад. Масалан:

```html
<html>
  <body>
    <h1>Hello world</h1>
    <p>Hello students</p>
  </body>
</html>
```

Сохтори дарахтмонанди он чунин аст:

- `html` — решаи дарахт.
- `body` — зернод.
- `h1` ва `p` — нодҳои кӯдакона.

### 2. Методҳои DOM

- `document.getElementById('id')`: Барои пайдо кардани унсури дорои ID муайян.
- `document.querySelector('selector')`: Барои интихоб кардани унсур бо селектори CSS.
- `element.textContent` ё `element.innerHTML`: Барои тағйир додани матн ё HTML-и дохилӣ.
- `createElement`, `appendChild`: Барои илова кардани унсурҳои нав.

### 4. Аҳамият

DOM яке аз асосҳои муҳим барои сохтани вебсайтҳои динамикӣ мебошад. Бо он мо метавонем бо интерфейси корбар кор карда, таҷрибаи корбариро беҳтар созем.

# InnerHtml

## innerHTML - ин хусусият (property) роҳи осонеро фароҳам меорад, ки унсури асосиро пурра иваз намояд. Масалан, тамоми мундариҷаи дохилии унсурро метавон хориҷ кард.

# DOM Events ва `onclick`

### DOM Events чист?

## DOM Events (ҳодисаҳои DOM) механизмест, ки ба саҳифаҳои веб имкон медиҳад, ки ба амалҳои корбар ё тағйирот дар ҳуҷҷат вокуниш нишон диҳанд. Ҳодисаҳо метавонанд дар натиҷаи:

- Амалҳои корбар (масалан, пахши тугма, ҳаракати муш, ворид кардани матн).
- Тағйирот дар ҳолати элемент (масалан, бор шудани тасвир, анҷоми анимация).
- Амалҳои браузер (масалан, пурра бор шудани саҳифа, хато дар скрипт).

### Навъҳои ҳодисаҳо дар DOM

#### 1. Ҳодисаҳои муш:

- `click` — пахш кардан бо муш.
- `dblclick` — ду маротиба пахш кардан.
- `mousedown` — пахш кардани тугмаи муш.
- `mouseup` — сар додани тугмаи муш.
- `mousemove` — ҳаракати муш.

#### 2. Ҳодисаҳои клавиатура:

- `keydown` — пахш кардани клавиша.
- `keyup` — сар додани клавиша.
- `keypress` — пахш кардани клавиша (сабт шудааст, аммо камтар истифода мешавад).

#### 3. Ҳодисаҳои форма:

- `submit` — фиристодани форма.
- `change` — тағйирот дар майдони воридот.
- `focus` — гузаштан ба элемент (масалан, майдони воридот).
- `blur` — баромадан аз элемент.

#### 4. Ҳодисаҳои ҳуҷҷат ва равзана:

- `load` — пурра бор шудани саҳифа.
- `resize` — тағйир додани андозаи равзана.
- `scroll` — ҳаракат дар дохили саҳифа.

### Событие `onclick` чист?

### `onclick` яке аз ҳодисаҳои маъруфтарин дар DOM мебошад, ки ҳангоми пахш кардани элемент бо муш ба кор медарояд. Онро метавон дар HTML ё JavaScript истифода бурд.

#### Мисоли истифода дар HTML:

```html
<button onclick="alert('Салом!')">Пахш кунед</button>
```

Дар ин ҷо, ҳангоми пахш кардани тугма паёми "Салом!" пайдо мешавад.

#### Мисоли истифода дар JavaScript:

```html
<button id="myButton">Пахш кунед</button>

<script>
  const button = document.getElementById("myButton");
  button.onclick = function () {
    alert("Салом аз JavaScript!");
  };
</script>
```

### Чӣ гуна истифода бурда мешавад?

- **Инлайн (дар HTML):** Барои ҳодисаҳои оддӣ.
- **Дар JavaScript:** Барои тақсим кардани функсияҳо ва нигоҳ доштани коди муназзам.

# `classList` чист?

## `classList` як хосияти (property) махсус дар DOM мебошад, ки барои кор бо рӯйхати классҳо (class-ҳо) дар элементҳои HTML истифода мешавад. Бо ёрии `classList` шумо метавонед бо классҳо бе зарурати дастӣ таҳрир кардани хосияти `className` кор кунед. Ин хосият пайвасти ҳамаи классҳои як элементро ҳамчун рӯйхат (DOMTokenList) пешниҳод мекунад.

### Методиҳои асосии `classList`

1. **`add()` — Илова кардани класс:**  
   Барои илова кардани як ё якчанд класс ба элемент истифода мешавад.  
   **Синтаксис:**

   ```javascript
   element.classList.add("class1", "class2");
   ```

   **Мисол:**

   ```javascript
   const myDiv = document.getElementById("myDiv");
   myDiv.classList.add("active", "highlight");
   ```

2. **`remove()` — Хориҷ кардани класс:**  
   Барои хориҷ кардани як ё якчанд класс аз элемент истифода мешавад.  
   **Синтаксис:**

   ```javascript
   element.classList.remove("class1", "class2");
   ```

   **Мисол:**

   ```javascript
   const myDiv = document.getElementById("myDiv");
   myDiv.classList.remove("active", "highlight");
   ```

3. **`toggle()` — Иваз кардани класс:**  
   Агар класс вуҷуд дошта бошад, онро хориҷ мекунад. Агар набошад, илова мекунад.  
   **Синтаксис:**
   ```javascript
   element.classList.toggle("className");
   ```
   **Мисол:**
   ```javascript
   const myDiv = document.getElementById("myDiv");
   myDiv.classList.toggle("visible");
   ```
